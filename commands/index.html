Liberty County Server Network Commands

Explore all commands available for your Discord server with Liberty County Server Network.

Tools Commands

mod - This command will ping the mod roles you provided on setup.
staff - This will ping the staff roles you provided on setup.
on - This will start the staff reminders.
off - This will stop your staff reminders.
embed - This will make an embed with your provided message.
say - This will send a message to the channel with the given message.
ad - This will send your advertisement to the channel.

Other Commands

setup - This will allow you to setup the bot to your liking.
settings - This will allow you to change some settings of the bot.
config - This will allow users to edit there configuration!
banners ssd - Search for 5 different ssd banners
banners sv - Search for 5 different svote banners
banners ssu - Search for 5 different ssu banners
info - This command will provide all important information about Liberty County Server Network.
help - Need help with commands? Run the help command.

Infractions Commands

promote - This command is used to promote any staff members who require a promotion.
warn - This command will warn a staff member.
search warns - This command will search any warns for a staff member.
search terminations - This command will search any terminations for a staff member.
search promotions - This command is used to search a user's promotions
search strikes - This command will search any strikes for a staff member.
search demotions - This command is used to search a user's demotions.
strike - This command will strike a staff member.
demote - This command is used to demote any staff members who require a demotion.
teminate - This command will terminate a staff member.
delete warn - This command will delete a warn for a staff member
delete strike> - This command will delete a strike for a staff member.

Session Commands

session shutdown - This command sends the Server Shutdown message.
session full - This command sends the Server Full message.
session restart - This command sends the Server Restart message.
session startup - This command sends the Server Start Up message.
session vcancel - This command sends the Session Cancel message.
session reminder - This command sends the Server Full message.
svote - This will start a session vote.

Logs Commands

search sts - This command will search any STS logs.
log ridealong - Create a ridealong log.
log sts - Create an STS log.
log prty - Create a priority log.
log message - Create a ridealong log.
